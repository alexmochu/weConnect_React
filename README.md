 # weConnect UI

> React front-end for [weConnect](#) Flask API.


[View on Heroku](https://weconnect-ui.herokuapp.com/)

## Install

1. Clone the repo: 
```
$ git clone [URL]
```
2. Install dependencies.
```
$ npm install
```
3. Run the app:
```
$ npm start
```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
## Running tests

```
$ npm test
```
